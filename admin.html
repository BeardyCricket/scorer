Beardy Cricket Admin
How to add a player
  1. Enter their name (exactly as typed in the scoring app) and email below → + Add
  2. Go to Supabase → Authentication → Users → Add user → Create new user
  3. Enter the same email and set a temporary password
  4. Share the stats.html URL and their temporary password with them
How to revoke access

Click Revoke next to a player below. They will be blocked immediately on their next login attempt. Their Supabase account still exists but the stats page won't let them in.

How to change a player's email
  1. Supabase → Table Editor → player_access — click their row and update the email
  2. Supabase → Authentication → Users — delete their old account, then Add user → Create new user with the new email
  3. Share the new login details with the player
